# PicTool
A camera tool that contains simple fx &amp; parades
## Build
```bash
> make
```
## Usage
Use default camera & default resolution (Maybe 640p)
```bash
> PicTool
```
Get camera count
```bash
> PicTool count
```
Start with specific camera & specific resolution
```bash
# PicTool <index> <width> <height>
> PicTool 0 1920 1080
```
